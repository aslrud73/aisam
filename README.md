# 오늘알림장 (Oneul Notification)

> 유치원·어린이집 선생님을 위한 AI 알림장 자동작성 앱

매일 20명의 학부모에게 보내는 개인별 알림장을 1~2시간씩 쓰는 일,
이제 AI가 도와드립니다. 선생님은 토글 몇 번과 짧은 메모만 남기시면
한 번 클릭으로 따뜻하고 개별성 있는 알림장이 모든 아이에 대해 동시에 완성됩니다.

## 왜 만들었나

한국 유치원·어린이집 교사의 가장 큰 페인 포인트 중 하나는 **알림장 작성**입니다.
키즈노트, 카카오톡, 문자 등으로 매일 보내는 개인별 일일 기록은
- 따뜻하고 개별성 있어야 하고
- 식사·기분·낮잠·활동 참여·교우관계·특이사항을 모두 담아야 하고
- 20명을 모두 다른 표현으로 써야 합니다

이 작업은 매일 1~2시간씩 소요되는 교사 번아웃의 대표적인 원인입니다.
**오늘알림장**은 이 일을 10분 안에 끝내드립니다.

## 사용 흐름

1. **반 아이들 등록** (한 번만, 이름 입력)
2. **오늘의 활동** 한 줄 입력 (예: "봄꽃 그리기 미술활동")
3. **아이별 빠른 입력** — 식사/기분/낮잠 토글 + 짧은 특이사항 메모 (한 명당 10초)
4. **클릭 한 번으로 일괄 생성**
5. **복사 → 키즈노트·카톡·문자에 붙여넣기**

## 기술

- **Next.js 14 + TypeScript + Tailwind CSS**
- **Claude Opus 4.7** (`@anthropic-ai/sdk`)
  - 적응형 사고 (`thinking: { type: "adaptive" }`)
  - 시스템 프롬프트 캐싱 (반복 호출 비용 절감)
  - 구조화된 JSON 스키마 출력 (안정적인 파싱)
- **localStorage** 기반 명단 저장 (별도 DB 없이도 동작)

## 로컬 실행

```bash
# 1. API 키 설정
cp .env.example .env.local
# .env.local 에 ANTHROPIC_API_KEY=sk-ant-... 입력

# 2. 의존성 설치 및 실행
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 접속.

## 프로젝트 구조

```
app/
├── layout.tsx         # 루트 레이아웃 (한국어 폰트)
├── page.tsx           # 메인 UI (5단계 플로우)
├── globals.css        # Tailwind + 한국어 폰트 (Gowun Dodum, Pretendard)
└── api/
    └── generate/
        └── route.ts   # Claude API 호출 (서버 라우트, API 키 보호)
```

## 향후 확장

- 사진/이미지 첨부 (Claude Vision으로 자동 캡션)
- 한 주/한 달 누적 관찰일지 자동 생성
- 학부모 상담 자료 자동 정리
- 키즈노트 API 연동 (직접 발송)
